# Limitless Wordle Clone

A full-featured Wordle clone built with **JavaScript**, **HTML**, **CSS**, and a **Node.js + Express + MongoDB backend**.

## Features

- Randomly selects a 5-letter solution from a MongoDB database.
- Interactive keyboard with color feedback (green, yellow, gray).
- Validates guesses with a flip animation.
- Tracks attempts and ends game with a win/loss message.
- Reset functionality to start a new game.
- Fully functional front-end and backend, easily run with a single command.

## Tech Stack

- **Frontend:** HTML, CSS, Vanilla JS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Dev Tools:** live-server, concurrently

## Installation

1. Clone the repository:
   git clone https://github.com/yourusername/limitless-wordle.git

2. Navigate to the project folder:
   cd limitless-wordle

3. Install dependencies:
   npm install

## Usage

Start both frontend and backend together:
npm start
